? A SWIFT code, also known as a Bank Identifier Code(BIC), is an international bank code that identifies particular banks worldwide. It's used to send/receive money to/from overseas banks. The bank branch/relationship manager will be able to provide you the SWIFT code for your bank account. Alternatively, you can also contact your banks FOREX department for the same. Please note the a SWIFT code is and 8 or 11 digit alphabetic code and differs from branch to branch for every bank. It is not the IFSC code.
